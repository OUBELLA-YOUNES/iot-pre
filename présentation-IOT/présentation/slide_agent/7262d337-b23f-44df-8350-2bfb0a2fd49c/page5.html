Architecture Globale du Système IoT Sécurisé

ESP32

Publie des données de capteurs via MQTT

Mosquitto

Broker centralisé recevant toutes les données MQTT

Suricata

Inspecte le trafic réseau pour détecter les intrusions

Node-RED

Automatise le flux de travail et traite les alertes

InfluxDB

Stocke les données d'intrusion en séries temporelles

Twilio

Envoie des alertes SMS en temps réel
