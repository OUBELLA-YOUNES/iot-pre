MQTT

Protocole de messagerie

Limitations

  • Simulation uniquement (pas de matériel physique réel)
  • Environnement de test contrôlé
  • Pas de protection active contre les attaques
  • Sécurité limitée à la détection et l'alerte

Focus de Détection

  • Attaques par force brute sur les identifiants MQTT
  • Trafic MQTT anormal et comportements suspects
  • Tentatives d'accès non autorisé aux topics MQTT
  • Injection de données malveillantes dans le flux MQTT
